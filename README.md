# Aula de Banco de Dados (Database Class)

Repositório criado para armazenar os scripts das aulas de banco de dados MySQL - Curso Tecnico em TI - Senac Tatuapé.

## Aula 1: Comandos Básicos de DDL (Data Definition Language)
Hoje aprendemos a criar estruturas básicas no MySQL:
- `CREATE DATABASE` e `CREATE TABLE` para estruturar dados.
- O uso de `VARCHAR` para textos e `TINYINT` para números inteiros pequenos (como idade).
- O comando `DESCRIBE` para visualizar as colunas criadas.
- Como deletar coisas usando o perigoso comando `DROP`.
